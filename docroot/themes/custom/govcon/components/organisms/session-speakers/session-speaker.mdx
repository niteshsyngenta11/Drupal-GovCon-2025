# Session Speaker

The **Session Speaker** component displays a structured list of speakers, each associated with sessions they are presenting.

## Usage

This component is primarily used to **showcase speakers** at a Author Detail Page. It dynamically renders a list of session associated with selected Author with details such as session title, status and conference year.

### Variables
- **`sessions`**: An array of session details associated with the speaker:
 - `title`: The session title.
 - `conference_year`: The year of the conference.
 - `status`: The sessionâ€™s acceptance status (e.g., "Accepted", "Backup").
