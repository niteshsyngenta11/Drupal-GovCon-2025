# Archive Session

The **Archive Session** component displays a collection of past event sessions in a structured and visually appealing format. Each session card includes essential details like the session title, speaker, date, and category.

## Usage

This component is primarily used in **conference or event archives** to allow users to browse past sessions effectively.
