# **Footer Component**

The **Footer** component is a structured section at the bottom of the site, containing branding, navigation links, social media icons, and a call-to-action (CTA) section.

## **Component Structure & Props**

### **Logo (`footer__logo`)**
The logo displayed in the footer section.

- **`src_path`** – The source URL for the logo image. _(Defaults to `Default.png` if not provided)_
- **`src_alt`** – Alternative text for accessibility. _(Defaults to `"Footer Logo"`)_

### **Footer Social Icons (`footer__social__icons`)**
The `footer__social__icons` block renders social media links.

- **`items`** – An array of social media menu items.

### **Quick Links (`footer__quick_links`)**
The `footer__quick_links` block displays a list of important links.

- **`items`** – An array of quick link menu items.

### **Footer Main Menu (`footer__main_menu`)**
The `footer__main_menu` block provides primary navigation options.

- **`items`** – An array of footer menu items.

### **Call-to-Action (CTA) (`footer__cta_card`)**
A section that provides additional information or an action link In Drupal this can be updated from **Configuration > GovCon Custom**.

- **`footer_description`** – A short text description. _(Optional)_
- **`footer_cta_url`** – The URL for the CTA button. _(Optional)_
- **`footer_cta_label`** – The label for the CTA button. _(Optional)_
