# **Colors**

The following colors are used throughout our site to maintain consistency in design and branding.

## **Implementation**

To apply colors, we provide predefined CSS classes for both text and background styling. Additionally, these classes are supported in Gutenberg.

### **Available CSS Classes**
- **`text-<color-name>`** – Sets the text color to the specified color.  
- **`bg-<color-name>`** – Sets the background color to the specified color.
