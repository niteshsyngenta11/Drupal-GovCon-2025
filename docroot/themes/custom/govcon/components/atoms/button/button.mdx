# Button

Buttons trigger actions in your application. Use them wisely.

## **Implementation**

The Button atom component accepts the following props similar to that of link with modifier props in addition:

- `link_url`: Specifies the target URL of the button.
- `link_target`: Sets the target of the button.
- `link_title`: Sets the title of the button.
- `link_content`: Sets the content of the button.
- `modifier` : Sets the Button Classes
