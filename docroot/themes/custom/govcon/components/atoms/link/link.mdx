# **Link**

The Link Atom component is utilized for rendering hyperlinks.

## **Implementation**

The Link atom component accepts the following props:

- `link_url`: Specifies the target URL of the hyperlink.
- `link_target`: Sets the target of the hyperlink.
- `link_title`: Sets the title of the hyperlink.
- `link_content`: Sets the content of the hyperlink.
- `link_classes` : Sets the Link Classes
