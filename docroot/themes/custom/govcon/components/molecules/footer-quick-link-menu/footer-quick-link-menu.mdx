# Footer Quick Links

The **Footer Quick Links** component displays a list of important links in the footer. It follows a structured format and allows customization using the `custom_class` prop.

## Component Preview

  <Story name="Footer Quick Links">

  </Story>

## Custom Class (`custom_class`)

The `custom_class` prop allows customization of the quick links menu styling.

### **Applied Classes**
| Class | Description |
|-------|------------|
| `"footer-quick-links body-text bg-dark"` | Default class for quick links, applying body text styles and a dark background. |

### **Twig Usage**
```twig
{% include '@molecules/footer-menu/footer-menu.twig' with { items: items, custom_class: 'footer-quick-links body-text bg-dark'} only %}
