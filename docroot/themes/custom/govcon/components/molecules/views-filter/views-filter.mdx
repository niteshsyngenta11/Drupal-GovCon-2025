# **View Form**

The **View Form** component allows users to filter and refine data views dynamically. It provides input fields, dropdowns, and other controls to modify displayed results.

## **Filter Options (`filter_options`)**

The `filter_options` prop allows users to define the available filtering criteria for the view.

### **Available Filter Options**
| Filter Option | Description |
|--------------|------------|
| `"text"` | Text input field for searching by keywords. |
| `"dropdown"` | Select dropdown for choosing categories. |
| `"checkbox"` | Checkbox options for filtering based on boolean values. |

## **Custom Classes (`classes`)**

The `classes` prop allows additional styling to be applied to the view form.

### **Applied Classes**
| Class | Description |
|-------|------------|
| `"views-filter"` | Default styling for the views filter form. |
