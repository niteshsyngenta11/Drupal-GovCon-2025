# **User Account Menu**

The **User Account Menu** component provides navigation options related to user authentication and account management. It typically includes links to login, profile, and account settings.

## **Component Preview**

  <Story name="User Account Menu">

  </Story>
</Canvas>

## **Menu Type (`menu_type`)**

The `menu_type` prop defines the type of menu being rendered.

### **Available Menu Types**
| Menu Type | Description |
|-----------|------------|
| `"user_account"` | Displays user-related navigation options, such as login and account settings. |

## **Custom Classes (`classes`)**

The `classes` prop allows adding custom styling to the user account menu.

### **Applied Classes**
| Class | Description |
|-------|------------|
| `"user-account-menu"` | Defines the default styling for the user account navigation. |
