# Footer Social Menu

The **Footer Social Menu** component displays a list of social media icons in the footer. It provides a flexible layout with a customizable class.

## Component Preview

<Canvas>
  <Story name="Footer Social Menu">
    {footerSocialMenuTwig({ items: footerSocialStories?.args?.items || [] })}
  </Story>
</Canvas>

## Custom Class (`custom_class`)

The `custom_class` prop applies specific styles to the social menu.

### **Applied Classes**
| Class | Description |
|-------|------------|
| `"footer-social-icons"` | Enables a flexible layout with spacing between icons. |

### **Twig Usage**
```twig
{% include '@molecules/footer-menu/footer-menu.twig' with { items: items, custom_class: 'footer-social-icons' } only %}
